# 依赖项
+ node ^v12.13.0
+ npm ^v6.13.0
+ 如果你先前将 gulp 旧版(v4以前)安装到全局环境中了，请执行 ```npm rm -g gulp```

# 安装
```
npm install -g kaci
npm install -g gulp-cli
```

# 初始化
```
kaci gulp           //空项目
npm install         //如为已存在的项目
```

# 启动本地服务
```
gulp start          //默认端口：1024
```

# 打包发布
```
gulp build
```
